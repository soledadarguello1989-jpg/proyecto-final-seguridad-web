# Proyecto final: Control de versiones y seguridad web

Este repositorio contiene una simulación de control de versiones y seguridad web en una aplicación transaccional.

El sistema simulado incluye:

- Login con usuario y contraseña.
- Validación de contraseña robusta.
- Código de verificación enviado al teléfono móvil.
- Registro de cambios mediante GitHub.
- Simulación de error y retorno a una versión anterior.

## Objetivo

Aplicar control de versiones con GitHub y buenas prácticas de seguridad web durante el desarrollo de una aplicación web.
